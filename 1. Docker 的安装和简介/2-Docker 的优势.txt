简要介绍 Docker 的优势


统一环境, 标准化部署
    Ubuntu CentOS 等服务器的安装运行环境是不一样的
    要使用某些新版的软件需要自行编译, 非常繁琐

解决复杂的依赖问题
    比如两个微服务 2 个不同的依赖互相冲突

隔离应用的运行环境
    比如 redis 获取服务器权限的漏洞

轻量级的虚拟环境, 相比虚拟机而言开销小速度快

统一的服务管理
    不同的服务有不同的管理工具和方式
    如果使用了 Docker 则统一用 Docker 管理

dockerhub 上有许多高价值的镜像可以直接使用
https://hub.docker.com/
